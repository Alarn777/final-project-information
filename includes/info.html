Search instructions

System supports any length boolean expressions, but (AND OR NOT () ) must be uppercase

System supports wildcard expressions. "ra*" in search will result in "rabbit,ramp,railway... Example

Admin panel is protected by password and will allow: activating/deactivating any file, force reindexing, loading files from source

Index is done with .pickle python module witch implements inverted index

The UI is done with the localhost server running on python

System supports crawler (in admin panel "Crawl for 100 files" button) implemented with "scrapy", runs on wikipedia.org

*Crawler can run only once. To enable crawler run again we need to restart the server.
